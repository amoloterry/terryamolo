Project Ideas

This website will showcase a GIS project focused on Assessing tree maturity for sustainable logging using LiDAR-derived tree height data in the Mount Kenya Forest.

Objectives

  • Develop a methodology for assessing tree maturity in the Mount Kenya Forest using LiDAR-derived tree height data
  • Evaluate the accuracy and effectiveness of LiDAR technology in determining tree maturity for sustainable logging practices.
  • . Provide recommendations for integrating LiDAR-based assessments into forest management policies and practices in Kenya.
